Simulador Web de Abastecimiento, Precios y Consumo Familiar

Proyecto desarrollado para Programación Web I. La aplicación permite analizar situaciones relacionadas con abastecimiento de carburantes, precios de alimentos, transporte, consumo familiar, escasez y pérdida del poder adquisitivo.

Explorar Escenarios

Contexto

Actualmente muchas familias enfrentan problemas relacionados con el aumento de precios, abastecimiento, transporte y disminución del poder adquisitivo. Este proyecto busca representar estos fenómenos mediante simuladores interactivos.
