# Simulador Web de Abastecimiento, Precios y Consumo Familiar

## Descripción

Este proyecto fue desarrollado para la materia Programación Web I.

La aplicación permite simular diferentes situaciones relacionadas con el abastecimiento de carburantes, incremento de precios, transporte, consumo familiar, escasez de productos y pérdida del poder adquisitivo mediante modelos matemáticos sencillos.

## Objetivo

Analizar el impacto económico de diferentes situaciones cotidianas utilizando simulaciones interactivas desarrolladas con tecnologías web.

## Tecnologías Utilizadas

* HTML5
* CSS3
* JavaScript
